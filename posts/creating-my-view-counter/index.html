Creating My View Counter

Sat, Sep 07, 2024

I love designing systems. It really tickles all the right parts of my brain. I'm using AWS to make my boring, static site into a more dynamic and interactive one.

This site was generated as a basic webpage with 11ty (which is fantastic, by the way). Originally, I just wanted a quick site for my writing and portfolio.

Somewhere along the way, though, I realized I could utilize Lambda and DynamoDB to really add some interactivity to the site, so I am slowly adding features here and there. The view counter on my portfolio page was a great place to start.

Overview

To create the view counter, I need a few things:

  • Javascript on the site that calls an API
  • An API
  • Some code to handle the API request
  • A database that the code can pull from
